# 橋梁点検PDF リンク追加ツール

橋梁定期点検PDFの「損傷図（その９）」と「損傷写真（その１０）」の間に
ナビゲーションボタンを自動追加するデスクトップアプリです。

**v8より径間番号ベースのシンプルなリンク方式に変更しました。**

## ダウンロード（ビルド済みアプリ）

右側の **Actions** タブ → 最新のワークフロー → **Artifacts** から
お使いのOS向けファイルをダウンロードしてください。

| OS | ファイル名 |
|---|---|
| Windows | `橋梁点検PDFリンク追加ツール-Windows.zip` |
| macOS | `橋梁点検PDFリンク追加ツール-Mac.zip` |

## 使い方

1. ダウンロードした `.exe`（Windows）または実行ファイル（Mac）を起動
2. PDFをドラッグ＆ドロップ
3. 「▶ 処理開始」をクリック

## 動作概要（v8）

径間番号が同じページ同士をボタンでリンクします。

- **損傷図の各ページ** → 同じ径間の損傷写真ページ全てへのボタン（青）
- **損傷写真の各ページ** → 同じ径間の損傷図ページ全てへのボタン（緑）

例：1径間の損傷図4枚・損傷写真8枚の場合
- 損傷図の各ページにボタン **8個**（損傷写真8ページへのリンク）
- 損傷写真の各ページにボタン **4個**（損傷図4ページへのリンク）

## 対応PDF

- 単一径間PDF
- 複数径間PDF（ヘッダーの径間番号を自動検出）

## v7からの変更点

| | v7 | v8 |
|---|---|---|
| リンク方式 | 損傷図内の「写真番号N-M」をOCRで読み取って対応付け | 径間番号が同じページ同士を全てリンク |
| OCR | 必要（Tesseract・Poppler） | **不要** |
| 必要ライブラリ | pikepdf / pypdf / pdf2image / pillow / pytesseract | **pikepdf / pypdf / pillow のみ** |
| 処理速度 | 遅い（OCR処理あり） | **高速** |
